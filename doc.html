All endpoints require JWT authentication for protected routes. Use Authorization: Bearer <token> header.

Authentication

POST /auth/login
Public

User login. Returns accessToken, refreshToken, user.

curl -X POST https://waste.qbrain.in/api/auth/login \ -H "Content-Type: application/json" \ -d '{"email":"admin@wastems.com","password":"admin123"}'
POST /auth/register
Public

User registration (if enabled).

curl -X POST https://waste.qbrain.in/api/auth/register \ -H "Content-Type: application/json" \ -d '{"email":"user@example.com","password":"Passw0rd!","name":"User","role":"citizen"}'
POST /auth/refresh
Public

Refresh access token.

curl -X POST https://waste.qbrain.in/api/auth/refresh \ -H "Content-Type: application/json" \ -d '{"refreshToken":""}'
GET /auth/me
admin, ulb_admin, supervisor

Get current user info.

curl https://waste.qbrain.in/api/auth/me \ -H "Authorization: Bearer "
